## 🧪 TESTE EM 3 ETAPAS

### PASSO 1️⃣: Departamento Submete

```
LOGIN:
  Email: departamento@empresa.com
  Senha: 123456

CLIQUE:
  Menu: 📤 Submissão

PREENCHA:
  Descrição: Conta de Luz
  Valor:     550.00
  Vencimento: 20/03/2026

CLIQUE:
  Botão: ✅ Enviar para Validação
  
RESULTADO:
  ✅ Requisição enviada para validação!
  (documento aparece na tabela com status PENDENTE ⏳)
```

---

### PASSO 2️⃣: Validador Aprova

```
SAIR:
  Clique no botão: 🚪 Sair

LOGIN:
  Email: validador@empresa.com
  Senha: 123456

CLIQUE:
  Menu: ✅ Validações

ENCONTRE:
  O documento "Conta de Luz" na tabela

CLIQUE:
  Botão: Aprovar
  
RESULTADO:
  ✅ Requisição aprovada!
  (status muda para VALIDADA 🟢)
```

---

### PASSO 3️⃣: Financeiro Paga

```
SAIR:
  Clique no botão: 🚪 Sair

LOGIN:
  Email: financeiro@empresa.com
  Senha: 123456

CLIQUE:
  Menu: 💳 Pagamentos

ENCONTRE:
  O documento "Conta de Luz" na tabela

CLIQUE:
  Botão: Processar Pagamento
  
RESULTADO:
  ✅ Pagamento realizado!
  (status muda para PAGA ✅)
```

## 🎨 CORES DOS STATUS

- 🟡 **PENDENTE** = Aguardando Validador
- 🟢 **VALIDADA** = Aguardando Financeiro
- 🔷 **PAGA** = Pronto ✅
- 🔴 **REJEITADA** = Volta ao Departamento

## 💡 IMPORTANTE

✅ **SIM, você pode:**
- Submeter vários documentos
- Ver histórico de tudo
- Acompanhar status em tempo real
- Testar com os 3 usuários diferentes

❌ **NÃO, você não pode:**
- Submeter como Validador ou Financeiro (menu não aparece)
- Pagar sem aprovação (só vê documentos VALIDADOS)
- Deletar documentos (fluxo é irreversível)
