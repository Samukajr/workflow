# Frontend - Sistema de Workflow de Pagamentos

## Descrição

Interface React + TypeScript para gerenciamento de fluxo de requisições de pagamento.

## Instalação

```bash
# Instalar dependências
npm install

# Copiar arquivo de exemplo de ambiente (se necessário)
cp .env.example .env
```

## Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`

## Build para Produção

```bash
npm run build
```

## Structure

- `/src/pages` - Páginas da aplicação
- `/src/components` - Componentes React reutilizáveis
- `/src/services` - Serviços de comunicação com a API
- `/src/store` - Estado global (Jotai)
- `/src/hooks` - Custom hooks
- `/src/types` - Definições de tipos TypeScript
- `/src/utils` - Funções utilitárias
